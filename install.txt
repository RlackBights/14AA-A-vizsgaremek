--- 1. install ---

- npm
    - Node.js letöltés, jön vele

- Android studio
    - Android studio letöltése a netről

- projekt

1. clone repo
2. temp folder (név mind1)
3. react install a temp folder-be: npx create-react-app my-app
4. be a temp projekt mappába: cd my-app
5. capacitor install: 
    - npm install --save-dev @iconify/react
    - npm install @capacitor/core @capacitor/cli
    - npx cap init my-app com.example.myapp --web-dir=build
    - npm i @capacitor/android
    - npx cap add android
    - npm i @capacitor-community/electron

6. a node-modules mappa másolása a vizsgaremek mappába

--- 2. test ---

szerver futtatása: npm run start

--- 3. build app ---

1. npm run build
2. npx cap open android
3. android studio-ban fenn
    - Build > Make Project
    - Build > Build Bundle(s) / APK(s) > Build APK(s)
4. a build a vizsgaremek\android\app\build\outputs\apk\debug mappában lesz

--- 4. build pc ---

1. npm run build
(-) npx cap add @capacitor-community/electron (csak egyszer!)
2. npx cap copy @capacitor-community/electron
3. cd electron
(-) npm install --save-dev @electron-forge/cli (csak egyszer!)
(-) npx electron-forge import (csak egyszer!)
4. npm run build
5. npx electron-forge package
